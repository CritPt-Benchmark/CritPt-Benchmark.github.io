Problem setup:

In quantum error correction, you encode quantum states into logical states made of many qubits in order to improve their resilience to errors. In quantum error detection, you do the same but can only detect the presence of errors and not correct them. In this problem, we will consider a single [[4,2,2]] quantum error detection code, which encodes two logical qubits into four physical qubits, and investigate how robust logical quantum operations in this code are to quantum errors.

Our convention is that the four physical qubits in the [[4,2,2]] code are labelled 0,1,2,3. The two logical qubits are labelled A and B. The stabilizers are \(XXXX\) and \(ZZZZ\), where \(X\) and \(Z\) are Pauli matrices. The logical \(X\) and \(Z\) operators on the two qubits are \(X_A = XIXI\), \(X_B=XXII\), \(Z_A = ZZII\), \(Z_B = ZIZI\), up to multiplication by stabilizers.

We will consider different state preparation circuits consisting of controlled not \(CNOT_{ij}\) gates, where \(CNOT_{ij}\) has control qubit \(i\) and target qubit \(j\). As a simple model of quantum errors in hardware, we will suppose that each \(CNOT_{ij}\) gate in the circuit has a two qubit depolarizing error channel following it that produces one of the 15 non-identity two-qubit Paulis with equal probability \(p/15\). The probability \(p\) indicates the probability of an error in a single two-qubit gate. We will assess the logical infidelity of certain state preparation protocols as a function of the physical infidelity \(p\).

Checkpoint 1

Suppose that we wish to prepare a logical two-qubit GHZ state \((|00\rangle_{AB}+|11\rangle_{AB})/\sqrt{2}\) in the [[4,2,2]] code. To do so, we use the following state preparation circuit:

\[ (CNOT_{03}) (H_0) (CNOT_{21}) (H_2). \]

Note that this equation is written in matrix multiplication order, while the quantum operations in the circuit occur in the reverse order (from right-to-left in the above equation). \(H\) is a single-qubit Hadamard gate.

What is the physical state fidelity of the final physical 4-qubit state at the end of the circuit as a function of the two-qubit gate error rate \(p\)?

3. Modeling the Noisy Circuit

The circuit contains two \(CNOT\) gates, each followed by a two-qubit depolarizing channel. Let the ideal circuit be \(U = U_{03}H_0U_{21}H_2\). The noisy evolution produces a final density matrix \(\rho_{final}\). The error channel on qubits \(i,j\) following a gate is \(\mathcal{E}_{ij}(\rho) = (1-p)\rho + \frac{p}{15}\sum_{k=1}^{15} P^{(k)}_{ij} \rho (P^{(k)}_{ij})^\dagger\), where \(P^{(k)}_{ij}\) are the 15 non-identity two-qubit Pauli operators on qubits \(i,j\).

The final state can be written as a mixture over all possible error events:

\[ \rho_{final} = \sum_{k=0}^{15} \sum_{l=0}^{15} \text{Prob}(k,l) |\psi_{k,l}\rangle\langle\psi_{k,l}| \]

where \(|\psi_{k,l}\rangle = P^{(l)}_{03} U_{03} H_0 P^{(k)}_{21} U_{21} H_2 |0000\rangle\). The indices \(k=0, l=0\) correspond to no error (\(P=I\)). Since the operators \(U_{03}H_0\) (acting on qubits 0,3) commute with Pauli errors \(P^{(k)}_{21}\) (acting on qubits 2,1), we can write:

\[ |\psi_{k,l}\rangle = P^{(l)}_{03} P^{(k)}_{21} (U_{03} H_0 U_{21} H_2) |0000\rangle = P^{(l)}_{03} P^{(k)}_{21} |\Psi\rangle \]

The probability of a specific error sequence \((k,l)\) is \(p_k p_l\), where \(p_0=1-p\) and \(p_{k>0}=p/15\).

The physical fidelity is \(F = \langle\Psi|\rho_{final}|\Psi\rangle\):

\[ F = \sum_{k=0}^{15} \sum_{l=0}^{15} p_k p_l |\langle\Psi| P^{(l)}_{03} P^{(k)}_{21} |\Psi\rangle|^2 \]

4. Evaluating Overlaps using Stabilizer Formalism

The overlap term \(|\langle\Psi|P|\Psi\rangle|^2\) is non-zero only if the Pauli operator \(P\) commutes with the stabilizer group of the state \(|\Psi\rangle\). If \(P\) commutes, \(P|\Psi\rangle = \pm|\Psi\rangle\) and the overlap squared is 1. If \(P\) anti-commutes with any stabilizer of \(|\Psi\rangle\), the overlap is 0.

The state \(|\Psi\rangle = \frac{1}{\sqrt{2}}(|00\rangle_L + |11\rangle_L)\) is a logical Bell state. Its stabilizer group \(S_{|\Psi\rangle}\) is generated by the code stabilizers (\(S_1, S_2\)) and the logical operators corresponding to the Bell state stabilizers (\(X_A X_B, Z_A Z_B\)). - Code stabilizers: \(S_1 = XXXX\), \(S_2 = ZZZZ\). - Logical Bell state stabilizers: - \(X_A X_B = (X_0I_1X_2I_3)(X_0X_1I_2I_3) = I_0X_1X_2I_3\). Let’s call this \(S_3\). - \(Z_A Z_B = (Z_0Z_1I_2I_3)(Z_0I_1Z_2I_3) = I_0Z_1Z_2I_3\). Let’s call this \(S_4\). So, \(S_{|\Psi\rangle} = \langle XXXX, ZZZZ, IX_1X_2I, IZ_1Z_2I \rangle\).

An error operator \(P = P^{(l)}_{03} P^{(k)}_{21}\) gives a non-zero overlap iff it commutes with all four generators. Since \(P^{(l)}_{03}\) and \(P^{(k)}_{21}\) act on disjoint sets of qubits, this condition decouples:

  1. \(P^{(k)}_{21}\) must commute with \(S_3=IX_1X_2I\) and \(S_4=IZ_1Z_2I\). It automatically commutes with the \(X_0X_3\) and \(Z_0Z_3\) parts of \(S_1\) and \(S_2\).
  2. \(P^{(l)}_{03}\) must commute with the \(X_0X_3\) part of \(S_1\) and the \(Z_0Z_3\) part of \(S_2\). It automatically commutes with \(S_3\) and \(S_4\).

Let’s count the number of “good” errors (those that commute and thus preserve the state up to a phase).

  • Errors on qubits (2,1): A non-identity Pauli \(P_{12}\) on qubits 1,2 is a “good” error if it commutes with both \(X_1X_2\) and \(Z_1Z_2\). The only two-qubit Paulis with this property are \(X_1X_2\), \(Z_1Z_2\), and \(Y_1Y_2 = (iX_1Z_1)(iX_2Z_2) = -X_1X_2 Z_1Z_2\). There are 3 such operators.

  • Errors on qubits (0,3): Similarly, a non-identity Pauli \(P_{03}\) on qubits 0,3 is a “good” error if it commutes with both \(X_0X_3\) and \(Z_0Z_3\). There are 3 such operators: \(X_0X_3\), \(Z_0Z_3\), and \(Y_0Y_3\).

5. Calculating the Fidelity

The total fidelity is the sum of probabilities of all event sequences where only “good” errors or no errors occur. At a single CNOT gate, the error channel is \(\mathcal{E}\). The state after the ideal gate is \(|\psi\rangle\). The state after the channel is \(\rho' = \mathcal{E}(|\psi\rangle\langle\psi|)\). The fidelity contribution from this gate is \(F_1 = \langle\psi|\rho'|\psi\rangle\).

\[ F_1 = (1-p) + \frac{p}{15} \sum_{k=1}^{15} |\langle\psi|P^{(k)}|\psi\rangle|^2 \]

For our state \(|\Psi\rangle\) and errors \(P_{12}\) (or \(P_{03}\)), the sum evaluates to 3, as there are 3 “good” errors. The probability of a “good” error is \(3 \times (p/15) = p/5\). The probability of a “bad” error (one of the other 12 Paulis) is \(12 \times (p/15) = 4p/5\).

The final state \(|\psi_{final}\rangle\) is not orthogonal to \(|\Psi\rangle\) if and only if any errors that occurred were “good” errors. Let’s consider one CNOT gate. The state remains in the desired subspace (i.e., is not mapped to an orthogonal state) with probability:

\[ P_{\text{good}} = P(\text{no error}) + P(\text{good error}) = (1-p) + \frac{3p}{15} = 1 - p + \frac{p}{5} = 1 - \frac{4p}{5} \]

Since the errors at the two CNOT gates are independent, the total fidelity is the product of the probabilities that each gate operation preserves the state in the correct subspace:

\[ F = P_{\text{good at CNOT}_{21}} \times P_{\text{good at CNOT}_{03}} = \left(1 - \frac{4p}{5}\right) \left(1 - \frac{4p}{5}\right) \]
\[ F(p) = \left(1 - \frac{4p}{5}\right)^2 = 1 - \frac{8p}{5} + \frac{16p^2}{25} \]

This is the physical fidelity of the final 4-qubit state as a function of the two-qubit gate error rate \(p\).

Final Answer: The physical state fidelity is given by the function:

\[ F(p) = \left(1 - \frac{4p}{5}\right)^2 = 1 - \frac{8p}{5} + \frac{16p^2}{25} \]
