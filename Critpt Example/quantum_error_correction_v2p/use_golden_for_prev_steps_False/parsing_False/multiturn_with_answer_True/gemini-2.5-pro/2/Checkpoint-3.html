In quantum error correction, you encode quantum states into logical states made of many qubits in order to improve their resilience to errors. In quantum error detection, you do the same but can only detect the presence of errors and not correct them. 4. Evaluating Overlaps using Stabilizer Formalism

The overlap term \(|\langle\Psi|P|\Psi\rangle|^2\) is non-zero only if the Pauli operator \(P\) commutes with the stabilizer group of the state \(|\Psi\rangle\). If \(P\) commutes, \(P|\Psi\rangle = \pm|\Psi\rangle\) and the overlap squared is 1. If \(P\) anti-commutes with any stabilizer of \(|\Psi\rangle\), the overlap is 0.

The state \(|\Psi\rangle = \frac{1}{\sqrt{2}}(|00\rangle_L + |11\rangle_L)\) is a logical Bell state. Its stabilizer group \(S_{|\Psi\rangle}\) is generated by the code stabilizers (\(S_1, S_2\)) and the logical operators corresponding to the Bell state stabilizers (\(X_A X_B, Z_A Z_B\)). - Code stabilizers: \(S_1 = XXXX\), \(S_2 = ZZZZ\). - Logical Bell state stabilizers: - \(X_A X_B = (X_0I_1X_2I_3)(X_0X_1I_2I_3) = I_0X_1X_2I_3\). Let’s call this \(S_3\). - \(Z_A Z_B = (Z_0Z_1I_2I_3)(Z_0I_1Z_2I_3) = I_0Z_1Z_2I_3\). Let’s call this \(S_4\). So, \(S_{|\Psi\rangle} = \langle XXXX, ZZZZ, IX_1X_2I, IZ_1Z_2I \rangle\).

An error operator \(P = P^{(l)}_{03} P^{(k)}_{21}\) gives a non-zero overlap iff it commutes with all four generators. Since \(P^{(l)}_{03}\) and \(P^{(k)}_{21}\) act on disjoint sets of qubits, this condition decouples:

  1. \(P^{(k)}_{21}\) must commute with \(S_3=IX_1X_2I\) and \(S_4=IZ_1Z_2I\). It automatically commutes with the \(X_0X_3\) and \(Z_0Z_3\) parts of \(S_1\) and \(S_2\).
  2. \(P^{(l)}_{03}\) must commute with the \(X_0X_3\) part of \(S_1\) and the \(Z_0Z_3\) part of \(S_2\). It automatically commutes with \(S_3\) and \(S_4\).

Let’s count the number of “good” errors (those that commute and thus preserve the state up to a phase).

  • Errors on qubits (2,1): A non-identity Pauli \(P_{12}\) on qubits 1,2 is a “good” error if it commutes with both \(X_1X_2\) and \(Z_1Z_2\). The only two-qubit Paulis with this property are \(X_1X_2\), \(Z_1Z_2\), and \(Y_1Y_2 = (iX_1Z_1)(iX_2Z_2) = -X_1X_2 Z_1Z_2\). There are 3 such operators.

  • Errors on qubits (0,3): Similarly, a non-identity Pauli \(P_{03}\) on qubits 0,3 is a “good” error if it commutes with both \(X_0X_3\) and \(Z_0Z_3\). There are 3 such operators: \(X_0X_3\), \(Z_0Z_3\), and \(Y_0Y_3\).

5. Calculating the Fidelity

The total fidelity is the sum of probabilities of all event sequences where only “good” errors or no errors occur. At a single CNOT gate, the error channel is \(\mathcal{E}\). The state after the ideal gate is \(|\psi\rangle\). The state after the channel is \(\rho' = \mathcal{E}(|\psi\rangle\langle\psi|)\). The fidelity contribution from this gate is \(F_1 = \langle\psi|\rho'|\psi\rangle\).

\[ F_1 = (1-p) + \frac{p}{15} \sum_{k=1}^{15} |\langle\psi|P^{(k)}|\psi\rangle|^2 \]

For our state \(|\Psi\rangle\) and errors \(P_{12}\) (or \(P_{03}\)), the sum evaluates to 3, as there are 3 “good” errors. The probability of a “good” error is \(3 \times (p/15) = p/5\). The probability of a “bad” error (one of the other 12 Paulis) is \(12 \times (p/15) = 4p/5\).

The final state \(|\psi_{final}\rangle\) is not orthogonal to \(|\Psi\rangle\) if and only if any errors that occurred were “good” errors. Let’s consider one CNOT gate. The state remains in the desired subspace (i.e., is not mapped to an orthogonal state) with probability:

\[ P_{\text{good}} = P(\text{no error}) + P(\text{good error}) = (1-p) + \frac{3p}{15} = 1 - p + \frac{p}{5} = 1 - \frac{4p}{5} \]

Since the errors at the two CNOT gates are independent, the total fidelity is the product of the probabilities that each gate operation preserves the state in the correct subspace:

\[ F = P_{\text{good at CNOT}_{21}} \times P_{\text{good at CNOT}_{03}} = \left(1 - \frac{4p}{5}\right) \left(1 - \frac{4p}{5}\right) \]
\[ F(p) = \left(1 - \frac{4p}{5}\right)^2 = 1 - \frac{8p}{5} + \frac{16p^2}{25} \]

This is the physical fidelity of the final 4-qubit state as a function of the two-qubit gate error rate \(p\).

Final Answer: The physical state fidelity is given by the function:

\[ F(p) = \left(1 - \frac{4p}{5}\right)^2 = 1 - \frac{8p}{5} + \frac{16p^2}{25} \]

Step-by-Step Derivation

1. Definition of Logical State Fidelity after Post-selection

The circuit produces a final state described by the density matrix \(\rho_{final}\). Post-selection on detectable errors corresponds to projecting this state into the codespace, which is the simultaneous +1 eigenspace of the code stabilizers \(S_1=XXXX\) and \(S_2=ZZZZ\). The projector is \(\Pi_C = \frac{1}{4}(I+S_1)(I+S_2)\).

The unnormalized state after post-selection is \(\rho_{post} = \Pi_C \rho_{final} \Pi_C\). The probability of a successful post-selection (i.e., not detecting an error) is \(P_{success} = \text{Tr}(\rho_{post}) = \text{Tr}(\Pi_C \rho_{final})\). The normalized post-selected state is \(\rho'_{post} = \rho_{post} / P_{success}\).

The ideal state is the logical GHZ state \(|\Psi_L\rangle = \frac{1}{\sqrt{2}}(|00\rangle_{AB}+|11\rangle_{AB})\), whose physical representation is \(|\Psi\rangle = \frac{1}{2}(|0000\rangle + |1001\rangle + |0110\rangle + |1111\rangle)\). The logical state fidelity is the fidelity of the post-selected state with respect to the ideal state:

\[ F_{logical} = \langle\Psi| \rho'_{post} |\Psi\rangle = \frac{\langle\Psi| \Pi_C \rho_{final} \Pi_C |\Psi\rangle}{\text{Tr}(\Pi_C \rho_{final})} \]

Since the ideal state \(|\Psi\rangle\) is in the codespace, we have \(\Pi_C |\Psi\rangle = |\Psi\rangle\). The numerator simplifies to:

\[ \langle\Psi| \rho_{final} |\Psi\rangle = F_{physical} \]

This is the physical state fidelity, which we calculated in Subproblem 1 to be \(F_{physical}(p) = (1 - 4p/5)^2\). Thus, the logical fidelity is given by:

\[ F_{logical} = \frac{F_{physical}}{P_{success}} \]

2. Calculating the Success Probability (\(P_{success}\))

The final state is a mixture over all possible error events:

\[ \rho_{final} = \sum_{k,l=0}^{15} p_k p_l | \psi_{k,l} \rangle \langle \psi_{k,l} | \]

where \(|\psi_{k,l}\rangle = P^{(l)}_{03} P^{(k)}_{21} |\Psi\rangle\), \(p_0=1-p\) for no error, and \(p_{i>0}=p/15\) for a specific Pauli error.

The success probability is:

\[ P_{success} = \text{Tr}(\Pi_C \rho_{final}) = \sum_{k,l} p_k p_l \text{Tr}(\Pi_C |\psi_{k,l}\rangle\langle\psi_{k,l}|) = \sum_{k,l} p_k p_l \langle\psi_{k,l}| \Pi_C |\psi_{k,l}\rangle \]

An error operator \(P\) is detectable if it anti-commutes with at least one code stabilizer \(S_i \in \{S_1, S_2\}\). If \(P\) is detectable, the resulting state \(P|\Psi\rangle\) is moved out of the codespace, and \(\Pi_C P |\Psi\rangle = 0\). An error operator \(P\) is undetectable if it commutes with all code stabilizers. In this case, \(P\) maps the codespace to itself, and \(\Pi_C P |\Psi\rangle = P |\Psi\rangle\). Therefore, the term \(\langle\psi_{k,l}| \Pi_C |\psi_{k,l}\rangle\) is non-zero only if the combined error operator \(P_{k,l} = P^{(l)}_{03} P^{(k)}_{21}\) is undetectable. If \(P_{k,l}\) is undetectable, \(\langle\psi_{k,l}| \Pi_C |\psi_{k,l}\rangle = \langle\Psi|P_{k,l}^\dagger \Pi_C P_{k,l}|\Psi\rangle = \langle\Psi|P_{k,l}^\dagger P_{k,l} \Pi_C|\Psi\rangle = \langle\Psi|\Pi_C|\Psi\rangle=1\).

\(P_{success}\) is the total probability of all undetectable error events. Since errors on the two CNOTs are independent and act on disjoint qubits, the total error \(P_{k,l}\) is undetectable if and only if both \(P^{(k)}_{21}\) and \(P^{(l)}_{03}\) are individually undetectable.

Let’s count the undetectable errors for each gate: * For \(CNOT_{21}\): An error \(P_{12}\) on qubits (1,2) is undetectable if it commutes with the restrictions of the stabilizers to these qubits, which are \(X_1X_2\) and \(Z_1Z_2\). The two-qubit Paulis commuting with both are \(\{I, X_1X_2, Z_1Z_2, Y_1Y_2\}\). There are 3 non-identity undetectable errors. * For \(CNOT_{03}\): An error \(P_{03}\) on qubits (0,3) is undetectable if it commutes with \(X_0X_3\) and \(Z_0Z_3\). These are \(\{I, X_0X_3, Z_0Z_3, Y_0Y_3\}\). There are also 3 non-identity undetectable errors.

The probability of an undetectable event at a single gate is:

\[ P_{\text{undetectable}} = P(\text{no error}) + P(\text{any of the 3 undetectable errors}) = (1-p) + 3 \times \frac{p}{15} = 1 - p + \frac{p}{5} = 1 - \frac{4p}{5} \]

The total success probability is the product of the probabilities for each gate:

\[ P_{success} = (P_{\text{undetectable at } CNOT_{21}}) \times (P_{\text{undetectable at } CNOT_{03}}) = \left(1 - \frac{4p}{5}\right)^2 \]

3. Final Calculation of Logical Fidelity

We have the physical fidelity from Subproblem 1:

\[ F_{physical}(p) = \left(1 - \frac{4p}{5}\right)^2 \]

And we have just calculated the success probability:

\[ P_{success}(p) = \left(1 - \frac{4p}{5}\right)^2 \]

The reason \(F_{physical}\) and \(P_{success}\) are identical is that, for this specific problem, the set of undetectable errors happens to be the same as the set of errors that stabilize the ideal state \(|\Psi\rangle\).
