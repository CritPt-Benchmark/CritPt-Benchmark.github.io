Problem setup:

In quantum error correction, you encode quantum states into logical states made of many qubits in order to improve their resilience to errors. In quantum error detection, you do the same but can only detect the presence of errors and not correct them. In this problem, we will consider a single [[4,2,2]] quantum error detection code, which encodes two logical qubits into four physical qubits, and investigate how robust logical quantum operations in this code are to quantum errors.

Our convention is that the four physical qubits in the [[4,2,2]] code are labelled 0,1,2,3. The two logical qubits are labelled A and B. The stabilizers are \(XXXX\) and \(ZZZZ\), where \(X\) and \(Z\) are Pauli matrices. The logical \(X\) and \(Z\) operators on the two qubits are \(X_A = XIXI\), \(X_B=XXII\), \(Z_A = ZZII\), \(Z_B = ZIZI\), up to multiplication by stabilizers.

We will consider different state preparation circuits consisting of controlled not \(CNOT_{ij}\) gates, where \(CNOT_{ij}\) has control qubit \(i\) and target qubit \(j\). As a simple model of quantum errors in hardware, we will suppose that each \(CNOT_{ij}\) gate in the circuit has a two qubit depolarizing error channel following it that produces one of the 15 non-identity two-qubit Paulis with equal probability \(p/15\). The probability \(p\) indicates the probability of an error in a single two-qubit gate. We will assess the logical infidelity of certain state preparation protocols as a function of the physical infidelity \(p\).

Challenge

Suppose that we prepare a logical two-qubit \(|00\rangle_{AB}\) state in the [[4,2,2]] code. To do so, we introduce an ancilla qubit, qubit 4, and use the following state preparation circuit:

\[ M_4 (CNOT_{04}) (CNOT_{34}) (CNOT_{23}) (CNOT_{10}) (CNOT_{12}) (H_1) \]

Note that this equation is written in matrix multiplication order, while the quantum operations in the circuit occur in the reverse order (from right-to-left in the above equation). \(H\) is a single-qubit Hadamard gate and \(M\) is a single-qubit measurement. The ancilla is used to detect errors in the state preparation circuit and makes the circuit fault-tolerant. If the ancilla measurement is \(|0\rangle\) (\(|1\rangle\)), the state preparation succeeds (fails).

What is the logical state fidelity of the final 2-qubit logical state at the end of the circuit as a function of two-qubit gate error rate \(p\), assuming the state is post-selected on all detectable errors in the code and on the ancilla qubit measuring \(|0\rangle\)?

2. Analysis of the Error Detection Mechanism

The problem states that we post-select on two conditions:

  1. The final state of qubits 0-3 is in the codespace (i.e., it is a +1 eigenstate of \(S_1\) and \(S_2\)).
  2. The ancilla qubit measurement yields \(|0\rangle\).

Let’s analyze the ancilla measurement. The unitary transformation due to the final two CNOTs on an arbitrary state \(|\psi\rangle_c |0\rangle_a\) is \(U_{anc} = CNOT_{04}CNOT_{34}\). This transforms the state to \(\frac{I+Z_0Z_3}{2}|\psi\rangle_c|0\rangle_a + \frac{I-Z_0Z_3}{2}|\psi\rangle_c|1\rangle_a\). Measuring the ancilla in the Z-basis is equivalent to a projective measurement of the operator \(Z_0Z_3\) on the code qubits. The outcome \(|0\rangle\) corresponds to projecting the code state onto the +1 eigenspace of \(Z_0Z_3\).

The operator \(Z_0Z_3\) commutes with both stabilizers: \([Z_0Z_3, S_1] = [Z_0Z_3, XXXX] = 0\), because \(Z_0\) anticommutes with \(X_0\) and \(Z_3\) anticommutes with \(X_3\) (two anticommutations result in commutation). \([Z_0Z_3, S_2] = [Z_0Z_3, ZZZZ] = 0\), because all Pauli Z operators commute. Since \(Z_0Z_3\) commutes with the stabilizers, it is a logical operator. Let’s identify which one. The logical operators given are \(X_A, X_B, Z_A, Z_B\). We find that:

\[ Z_A Z_B S_2 = (Z_0Z_1I_2I_3)(Z_0I_1Z_2I_3)(Z_0Z_1Z_2Z_3) = (Z_0^2 Z_1 I_2 Z_2 I_3)(Z_0Z_1Z_2Z_3) \]
\[ = (I_0 Z_1 Z_2 I_3)(Z_0Z_1Z_2Z_3) = Z_0 Z_1^2 Z_2^2 Z_3 = Z_0 I_1 I_2 Z_3 = Z_0Z_3 \]

On the codespace, where \(S_2\) acts as the identity, the check operator \(Z_0Z_3\) is equivalent to the logical operator \(Z_A Z_B\).

Therefore, the post-selection criteria are that the final state must be a +1 eigenstate of \(S_1, S_2\), and the logical operator \(Z_A Z_B\).

3. First-Order Error Analysis

We consider the effect of a single two-qubit Pauli error, which occurs with probability \(p/15\) after one of the 5 CNOT gates. We need to calculate the fidelity of the final state, conditioned on passing all checks. The fidelity of a final state \(\rho_f\) with respect to the ideal state \(|\overline{00}\rangle\) is \(F = \langle\overline{00}|\rho_f|\overline{00}\rangle\). Infidelity is \(1-F\).

To first order in \(p\), the infidelity is the sum of probabilities of all single-error events that pass the checks and result in a state orthogonal to \(|\overline{00}\rangle\).

\[ 1 - F = \sum_{\text{errors } E} P(E) \times \mathbb{I}(E \text{ is not detected}) \times (1 - |\langle\overline{00}|\psi_E\rangle|^2) + O(p^2) \]

where \(|\psi_E\rangle\) is the final state when error \(E\) occurs.

An error is not detected if the state it produces passes the checks. Let a single physical Pauli error \(P\) occur at some point in the circuit. This is equivalent to a logical Pauli operator \(L\) acting on the ideal final state \(|\overline{00}\rangle\). The final state is \(L|\overline{00}\rangle\). For this error to be undetected, the state \(L|\overline{00}\rangle\) must be a +1 eigenstate of \(S_1, S_2\), and \(Z_A Z_B\). This implies that the error operator \(L\) must commute with \(S_1, S_2\), and \(Z_A Z_B\).

The fidelity of the resulting state is \(F_L = |\langle\overline{00}|L|\overline{00}\rangle|^2\). The ideal state \(|\overline{00}\rangle\) is a logical Z-basis state. Its fidelity is only reduced by logical X-type or Y-type errors, which would flip it to an orthogonal logical state (e.g., \(X_A|\overline{00}\rangle = |\overline{10}\rangle\), for which \(\langle\overline{00}|\overline{10}\rangle=0\)). Logical Z-type errors leave the state \(|\overline{00}\rangle\) unchanged (up to a phase), so they do not reduce fidelity. \(Z_A|\overline{00}\rangle = |\overline{00}\rangle\) and \(Z_B|\overline{00}\rangle = |\overline{00}\rangle\).

The crucial step is to determine which logical operators are detected by the ancilla check (i.e., anticommute with \(Z_A Z_B\)). * Logical X operators: * \([X_A, Z_A Z_B] = X_A Z_A Z_B - Z_A Z_B X_A = ( -Z_A X_A) Z_B - Z_A (Z_B X_A) = -Z_A X_A Z_B - Z_A X_A Z_B = -2Z_A X_A Z_B \neq 0\). \(X_A\) anticommutes with \(Z_A\) and commutes with \(Z_B\). Thus, \(\{X_A, Z_A Z_B\} = 0\). So \(X_A\) is detected. * \([X_B, Z_A Z_B] \neq 0\). \(X_B\) commutes with \(Z_A\) and anticommutes with \(Z_B\). Thus, \(\{X_B, Z_A Z_B\} = 0\). So \(X_B\) is detected. * Logical Y operators: * \(Y_A = iX_AZ_A\). \(Y_A\) anticommutes with \(Z_B\) and commutes with \(Z_A\). So \(\{Y_A, Z_A Z_B\} = 0\). \(Y_A\) is detected. * \(Y_B = iX_BZ_B\). \(Y_B\) commutes with \(Z_B\) and anticommutes with \(Z_A\). So \(\{Y_B, Z_A Z_B\} = 0\). \(Y_B\) is detected. * Logical Z operators: * \(Z_A\) commutes with both \(Z_A\) and \(Z_B\). So \([Z_A, Z_A Z_B] = 0\). \(Z_A\) is not detected by the ancilla. * \(Z_B\) commutes with both \(Z_A\) and \(Z_B\). So \([Z_B, Z_A Z_B] = 0\). \(Z_B\) is not detected by the ancilla.

This analysis shows that any single physical error that propagates to a logical error of type X or Y will be detected by the ancilla measurement. The only logical errors that can pass all checks are Z-type logical errors.

However, a logical Z error acting on the prepared state \(|\overline{00}\rangle\) does not change the state, since \(|\overline{00}\rangle\) is a +1 eigenstate of both \(Z_A\) and \(Z_B\).

\[ Z_A|\overline{00}\rangle = |\overline{00}\rangle, \quad Z_B|\overline{00}\rangle = |\overline{00}\rangle, \quad Z_A Z_B|\overline{00}\rangle = |\overline{00}\rangle \]

For any undetectable logical error \(L\), the final state is \(L|\overline{00}\rangle = |\overline{00}\rangle\). The fidelity contribution is \(F_L = |\langle\overline{00}|L|\overline{00}\rangle|^2 = |\langle\overline{00}|\overline{00}\rangle|^2 = 1\). The infidelity contribution is 0.

4. Conclusion

Any single two-qubit Pauli error occurring at any of the five CNOT gates leads to one of three outcomes:

  1. The error propagates to an operator that takes the state out of the codespace (e.g., anticommutes with \(S_1\) or \(S_2\)). This is detected by the code itself.
  2. The error propagates to a logical operator of X or Y type. This is detected by the ancilla measurement, which checks the logical operator \(Z_A Z_B\).
  3. The error propagates to a logical operator of Z type. This error is not detected, but it does not alter the target state \(|\overline{00}\rangle\).

In all cases where a single error occurs, either the error is detected (and the run is discarded), or the final state remains the correct logical state \(|\overline{00}\rangle\). Therefore, no single error can decrease the fidelity of the post-selected state. The logical infidelity is caused by events of two or more errors. Thus, the infidelity is of order \(O(p^2)\). To first order in \(p\), the infidelity is zero.

The logical state fidelity \(F\) is therefore:

\[ F(p) = 1 - O(p^2) \]

As a function of \(p\), to first order, the fidelity is 1.
