Problem setup:

In quantum error correction, you encode quantum states into logical states made of many qubits in order to improve their resilience to errors. In quantum error detection, you do the same but can only detect the presence of errors and not correct them. In this problem, we will consider a single [[4,2,2]] quantum error detection code, which encodes two logical qubits into four physical qubits, and investigate how robust logical quantum operations in this code are to quantum errors.

Our convention is that the four physical qubits in the [[4,2,2]] code are labelled 0,1,2,3. The two logical qubits are labelled A and B. The stabilizers are \(XXXX\) and \(ZZZZ\), where \(X\) and \(Z\) are Pauli matrices. The logical \(X\) and \(Z\) operators on the two qubits are \(X_A = XIXI\), \(X_B=XXII\), \(Z_A = ZZII\), \(Z_B = ZIZI\), up to multiplication by stabilizers.

We will consider different state preparation circuits consisting of controlled not \(CNOT_{ij}\) gates, where \(CNOT_{ij}\) has control qubit \(i\) and target qubit \(j\). As a simple model of quantum errors in hardware, we will suppose that each \(CNOT_{ij}\) gate in the circuit has a two qubit depolarizing error channel following it that produces one of the 15 non-identity two-qubit Paulis with equal probability \(p/15\). The probability \(p\) indicates the probability of an error in a single two-qubit gate. We will assess the logical infidelity of certain state preparation protocols as a function of the physical infidelity \(p\).

Challenge

Suppose that we prepare a logical two-qubit \(|00\rangle_{AB}\) state in the [[4,2,2]] code. To do so, we introduce an ancilla qubit, qubit 4, and use the following state preparation circuit:

\[ M_4 (CNOT_{04}) (CNOT_{34}) (CNOT_{23}) (CNOT_{10}) (CNOT_{12}) (H_1) \]

Note that this equation is written in matrix multiplication order, while the quantum operations in the circuit occur in the reverse order (from right-to-left in the above equation). \(H\) is a single-qubit Hadamard gate and \(M\) is a single-qubit measurement. The ancilla is used to detect errors in the state preparation circuit and makes the circuit fault-tolerant. If the ancilla measurement is \(|0\rangle\) (\(|1\rangle\)), the state preparation succeeds (fails).

What is the logical state fidelity of the final 2-qubit logical state at the end of the circuit as a function of two-qubit gate error rate \(p\), assuming the state is post-selected on all detectable errors in the code and on the ancilla qubit measuring \(|0\rangle\)?

The problem asks for the logical state fidelity of the prepared \(|00\rangle_{AB}\) state as a function of the two-qubit gate error rate \(p\). The state preparation involves a circuit with five CNOT gates, each followed by a two-qubit depolarizing error channel. The fidelity is to be calculated after post-selection on two conditions: the ancilla qubit (qubit 4) measures \(|0\rangle\), and the state on the four physical qubits (0,1,2,3) is in the code space (i.e., all stabilizers measure +1). We will calculate the fidelity to first order in \(p\).

1. Ideal State Preparation and Code Properties:

The initial state is \(|00000\rangle\). The circuit operations, in execution order (right-to-left as written), are: \(U_{circ} = CNOT_{04} \cdot CNOT_{34} \cdot CNOT_{23} \cdot CNOT_{10} \cdot CNOT_{12} \cdot H_1\).

Let’s trace the ideal state:

  1. \(|00000\rangle\)
  2. \(H_1\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |01000\rangle)\)
  3. \(CNOT_{12}\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |01100\rangle)\)
  4. \(CNOT_{10}\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |11100\rangle)\)
  5. \(CNOT_{23}\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |11110\rangle)\)
  6. \(CNOT_{34}\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |11111\rangle)\)
  7. \(CNOT_{04}\): \(\frac{1}{\sqrt{2}}(|00000\rangle + |11110\rangle)\)

The ideal final state, including the ancilla, is \(|\psi_{target}\rangle = \frac{1}{\sqrt{2}}(|00000\rangle + |11110\rangle)\). This state satisfies the ancilla measurement condition (\(M_4=|0\rangle\)) as both terms have qubit 4 in \(|0\rangle\). Now let’s check if the state on qubits 0,1,2,3, i.e., \(|\psi_{code}\rangle = \frac{1}{\sqrt{2}}(|0000\rangle + |1111\rangle)\), is the logical \(|00\rangle_{AB}\) state for the given code. The stabilizers are \(S_1 = X_0X_1X_2X_3\) and \(S_2 = Z_0Z_1Z_2Z_3\). The logical Z operators are \(Z_A = Z_0Z_1I_2I_3\) and \(Z_B = Z_0I_1Z_2I_3\). * \(S_1|\psi_{code}\rangle = X_0X_1X_2X_3 \frac{1}{\sqrt{2}}(|0000\rangle + |1111\rangle) = \frac{1}{\sqrt{2}}(|1111\rangle + |0000\rangle) = |\psi_{code}\rangle\). * \(S_2|\psi_{code}\rangle = Z_0Z_1Z_2Z_3 \frac{1}{\sqrt{2}}(|0000\rangle + |1111\rangle) = \frac{1}{\sqrt{2}}(|0000\rangle + (-1)^4|1111\rangle) = |\psi_{code}\rangle\). * \(Z_A|\psi_{code}\rangle = Z_0Z_1 \frac{1}{\sqrt{2}}(|0000\rangle + |1111\rangle) = \frac{1}{\sqrt{2}}(|0000\rangle + (-1)^2|1111\rangle) = |\psi_{code}\rangle\). * \(Z_B|\psi_{code}\rangle = Z_0Z_2 \frac{1}{\sqrt{2}}(|0000\rangle + |1111\rangle) = \frac{1}{\sqrt{2}}(|0000\rangle + (-1)^2|1111\rangle) = |\psi_{code}\rangle\). Since \(|\psi_{code}\rangle\) is a +1 eigenstate of \(S_1, S_2, Z_A, Z_B\), it is indeed the logical \(|00\rangle_{AB}\) state.

2. Fidelity Calculation to First Order in \(p\):

Each of the 5 CNOT gates (\(C_1=CNOT_{12}, C_2=CNOT_{10}, C_3=CNOT_{23}, C_4=CNOT_{34}, C_5=CNOT_{04}\)) is followed by a two-qubit depolarizing error channel. This means that after each CNOT, one of the 15 non-identity two-qubit Pauli operators \(P_k\) (acting on the CNOT’s qubits) occurs with probability \(p/15\). Let \(\rho_{ideal} = |\psi_{target}\rangle\langle\psi_{target}|\). The noisy state, to first order in \(p\), is:

\[ \rho_{noisy} \approx (1-5p) \rho_{ideal} + \sum_{m=1}^5 \sum_{k=1}^{15} \frac{p}{15} \rho_{m,k} \]

where \(\rho_{m,k} = |\psi_{m,k}\rangle\langle\psi_{m,k}|\) is the state obtained if the error \(P_k\) occurs after the \(m\)-th CNOT gate (denoted \(U_m\)), and all other gates are ideal. Let \(U_{before\_m}\) be the product of gates before \(U_m\), and \(U_{after\_m}\) be the product of gates after \(U_m\). So \(|\psi_{m,k}\rangle = U_{after\_m} P_k U_m U_{before\_m} |00000\rangle\).

The fidelity \(F\) is defined as \(F = \frac{\langle \psi_{target} | P_{succ\_proj} \rho_{noisy} P_{succ\_proj} | \psi_{target} \rangle}{\text{Tr}(P_{succ\_proj} \rho_{noisy})}\), where \(P_{succ\_proj} = P_S P_4^0\) projects onto the code space and ancilla \(|0\rangle\). Since \(P_{succ\_proj} |\psi_{target}\rangle = |\psi_{target}\rangle\), the numerator simplifies to \(\langle \psi_{target} | \rho_{noisy} | \psi_{target} \rangle\). Let \(N = \langle \psi_{target} | \rho_{noisy} | \psi_{target} \rangle\) and \(P_{succ} = \text{Tr}(P_{succ\_proj} \rho_{noisy})\).

\[ N \approx (1-5p) + \frac{p}{15} \sum_{m=1}^5 \sum_{k=1}^{15} |\langle \psi_{target} | \psi_{m,k} \rangle|^2 \]
\[ P_{succ} \approx (1-5p) + \frac{p}{15} \sum_{m=1}^5 \sum_{k=1}^{15} \|P_{succ\_proj} |\psi_{m,k}\rangle\|^2 \]

To first order in \(p\), the fidelity is \(F \approx 1 - \frac{p}{15} \sum_{m=1}^5 \sum_{k=1}^{15} \left( \|P_{succ\_proj} |\psi_{m,k}\rangle\|^2 - |\langle \psi_{target} | \psi_{m,k} \rangle|^2 \right)\).

Let’s classify each of the \(5 \times 15 = 75\) single error paths: * Ancilla detection (\(L_A\)): If the error \(P_k\) results in qubit 4 being \(|1\rangle\), then \(P_4^0 |\psi_{m,k}\rangle = 0\). This means \(\|P_{succ\_proj} |\psi_{m,k}\rangle\|^2 = 0\) and \(|\langle \psi_{target} | \psi_{m,k} \rangle|^2 = 0\). These errors contribute 0 to the sum. Errors involving \(X_4\) or \(Y_4\) as a component will flip qubit 4. There are 4 such Pauli errors for a single qubit (e.g., \(X_4, Y_4, X_0X_4, Y_0Y_4\)). The gates \(CNOT_{34}\) and \(CNOT_{04}\) affect qubit 4. For \(CNOT_{34}\), errors like \(I_3X_4, X_3X_4, Y_3X_4, Z_3X_4\) (4 types) and \(I_3Y_4, X_3Y_4, Y_3Y_4, Z_3Y_4\) (4 types) lead to ancilla failure. This is \(4+4=8\) errors. Similarly, for \(CNOT_{04}\), there are 8 errors leading to ancilla failure. Total \(N_A = 8+8=16\) errors are detected by the ancilla.

  • Stabilizer detection (\(L_D\)): If an error leads to a state not in the code space (i.e., a stabilizer measures -1), then \(P_S |\psi_{m,k}\rangle = 0\). These errors also contribute 0 to the sum. An error \(E\) is detected by stabilizers if it anti-commutes with at least one stabilizer. The code has distance \(d=2\), so all single-qubit errors (\(X_i, Y_i, Z_i\)) are detected.

  • Logical Z errors (\(L_Z\)): These are errors \(P_k\) that, after propagation, result in an effective operator \(L\) (on the data qubits) that maps the target state to itself. This includes \(I_L\), logical \(Z_A, Z_B, Z_AZ_B\), and any stabilizer product (e.g., \(S_1\)). For these errors, \(L|\psi_{code}\rangle = |\psi_{code}\rangle\). They yield \(\|P_{succ\_proj} |\psi_{m,k}\rangle\|^2 = 1\) and \(|\langle \psi_{target} | \psi_{m,k} \rangle|^2 = 1\). They contribute \(1-1=0\) to the sum for \(F\).

  • Logical X/Y errors (\(L_X\)): These are errors \(P_k\) that, after propagation, result in an effective operator \(L\) that maps the target state to an orthogonal logical state (e.g., \(X_A, Y_A, X_B, Y_B\), or combinations like \(X_AX_B\)). These errors commute with the stabilizers, so they are not detected. They yield \(\|P_{succ\_proj} |\psi_{m,k}\rangle\|^2 = 1\) but \(|\langle \psi_{target} | \psi_{m,k} \rangle|^2 = 0\). They contribute \(1-0=1\) to the sum for \(F\).

The fidelity formula simplifies to \(F \approx 1 - \frac{p}{15} N_X\), where \(N_X\) is the total count of errors that result in \(L_X\) type logical errors.

We need to propagate each non-\(L_A\) error through the rest of the circuit and determine its final effective action on the data qubits. Let \(P_k\) be an error after \(U_m\). The effective operator is \(E_{eff} = U_{after\_m} P_k U_{after\_m}^\dagger\).

Let’s list the 5 CNOT gates (\(U_m\)) and analyze their error propagation: \(U_1 = CNOT_{12}\) (on qubits 1,2). \(U_{after\_1} = CNOT_{04} \cdot CNOT_{34} \cdot CNOT_{23} \cdot CNOT_{10}\). \(U_2 = CNOT_{10}\) (on qubits 1,0). \(U_{after\_2} = CNOT_{04} \cdot CNOT_{34} \cdot CNOT_{23}\). \(U_3 = CNOT_{23}\) (on qubits 2,3). \(U_{after\_3} = CNOT_{04} \cdot CNOT_{34}\). \(U_4 = CNOT_{34}\) (on qubits 3,4). \(U_{after\_4} = CNOT_{04}\). \(U_5 = CNOT_{04}\) (on qubits 0,4). \(U_{after\_5} = I\).

3. Total Count and Final Fidelity:

Summing up the counts: Total \(N_X = 3+2+0+0+0 = 5\). Total \(N_Z = 3+1+2+1+1 = 8\). Total \(N_{L_A} = 16\). Total \(N_{L_D} = 75 - N_X - N_Z - N_{L_A} = 75 - 5 - 8 - 16 = 46\).

\[ F \approx 1 - \frac{p}{15} N_X \]
\[ F \approx 1 - \frac{p}{15} \times 5 \]
\[ F \approx 1 - \frac{5p}{15} \]
