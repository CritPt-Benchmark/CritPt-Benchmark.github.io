Problem setup:

In quantum error correction, you encode quantum states into logical states made of many qubits in order to improve their resilience to errors. In quantum error detection, you do the same but can only detect the presence of errors and not correct them. In this problem, we will consider a single [[4,2,2]] quantum error detection code, which encodes two logical qubits into four physical qubits, and investigate how robust logical quantum operations in this code are to quantum errors.

Our convention is that the four physical qubits in the [[4,2,2]] code are labelled 0,1,2,3. The two logical qubits are labelled A and B. The stabilizers are \(XXXX\) and \(ZZZZ\), where \(X\) and \(Z\) are Pauli matrices. The logical \(X\) and \(Z\) operators on the two qubits are \(X_A = XIXI\), \(X_B=XXII\), \(Z_A = ZZII\), \(Z_B = ZIZI\), up to multiplication by stabilizers.

We will consider different state preparation circuits consisting of controlled not \(CNOT_{ij}\) gates, where \(CNOT_{ij}\) has control qubit \(i\) and target qubit \(j\). As a simple model of quantum errors in hardware, we will suppose that each \(CNOT_{ij}\) gate in the circuit has a two qubit depolarizing error channel following it that produces one of the 15 non-identity two-qubit Paulis with equal probability \(p/15\). The probability \(p\) indicates the probability of an error in a single two-qubit gate. We will assess the logical infidelity of certain state preparation protocols as a function of the physical infidelity \(p\).

Challenge

Suppose that we prepare a logical two-qubit \(|00\rangle_{AB}\) state in the [[4,2,2]] code. To do so, we introduce an ancilla qubit, qubit 4, and use the following state preparation circuit:

\[ M_4 (CNOT_{04}) (CNOT_{34}) (CNOT_{23}) (CNOT_{10}) (CNOT_{12}) (H_1) \]

Note that this equation is written in matrix multiplication order, while the quantum operations in the circuit occur in the reverse order (from right-to-left in the above equation). \(H\) is a single-qubit Hadamard gate and \(M\) is a single-qubit measurement. The ancilla is used to detect errors in the state preparation circuit and makes the circuit fault-tolerant. If the ancilla measurement is \(|0\rangle\) (\(|1\rangle\)), the state preparation succeeds (fails).

What is the logical state fidelity of the final 2-qubit logical state at the end of the circuit as a function of two-qubit gate error rate \(p\), assuming the state is post-selected on all detectable errors in the code and on the ancilla qubit measuring \(|0\rangle\)?

  • Code and conventions:
  • [[4,2,2]] code with stabilizers S = ⟨XXXX, ZZZZ⟩ on physical qubits {0,1,2,3}.
  • Logical operators (up to stabilizers): X_A = X I X I, Z_A = Z Z I I; X_B = X X I I, Z_B = Z I Z I.
  • Circuit (right-to-left execution on qubits 0–4):
    • H on qubit 1
    • CNOT12, CNOT10, CNOT23, CNOT34, CNOT04
    • Measure ancilla 4 in Z (post-select outcome |0⟩)
  • Noise model: after each CNOT (there are 5), apply two-qubit depolarizing channel on that pair: with probability p a uniformly random non-identity 2-qubit Pauli on those two qubits (15 possibilities equally likely), and with probability 1−p no error.

  • Post-selection: discard runs with any detectable error by the code and any run where ancilla outcome is |1⟩.

  • Goal:

  • Compute logical state fidelity F_logical(p) = ⟨ψ_L| ρ_L |ψ_L⟩ as a function of p, where |ψ_L⟩ is the intended logical |00⟩_AB. Because we post-select on all detectable errors and the circuit is fault-tolerant via the ancilla, the first nonzero reduction in fidelity arises from error events that are not detected and that cause a logical error.

  • Error detectability in [[4,2,2]]:

  • The code detects any single-qubit Pauli error on the 4 data qubits (distance 2).
  • It detects any single-qubit Pauli on any one data qubit arising effectively from a single fault after error propagation.
  • However, some two-qubit error patterns from a single CNOT fault can commute with stabilizers and lie in the normalizer modulo stabilizer, thereby creating a logical operation (undetected) if they also avoid being flagged by the ancilla mechanism.

  • The ancilla is coupled by two CNOTs: CNOT34 and CNOT04, and then measured in Z. This provides an additional parity check to catch certain propagated faults (making the state-prep circuit first-order fault-tolerant: any single gate fault is either detected by the code syndrome or flips the ancilla to |1⟩).

  • Structure of first-order analysis:

  • Consider one fault (at most one two-qubit depolarizing error) among the 5 CNOTs. Probability of exactly one gate having an error ≈ 5 p to first order.
  • For each gate location ℓ ∈ {12,10,23,34,04}, and each non-identity 2-qubit Pauli E on that gate’s qubits, propagate E through the remainder of the circuit to just before the final checks (code detectability and ancilla Z measurement).
  • Classify outcomes:
    • Detected by ancilla (outcome 1): event is discarded.
    • Detected by code (nontrivial stabilizer syndrome if one measured): since we post-select on “all detectable errors in the code,” any event that would flip either stabilizer eigenvalue is discarded.
    • Undetected and no logical action (equivalent to a stabilizer): accepted, no fidelity loss.
    • Undetected logical error (element of normalizer not in stabilizer): accepted but wrong logical state; contributes to infidelity.
  • By circuit design, this gadget is fault-tolerant to first order: any single fault is either detected by the ancilla or detected by the code. Hence the probability of an undetected logical error from a single-fault event is zero.

  • Therefore, the logical infidelity begins at O(p^2).

  • Second-order contribution:

  • With two faults on two of the five CNOTs (or two-Pauli components within the same depolarizing instance across the same pair are not in this model), the joint error can conspire to evade both the ancilla and the code detectability while inducing a logical operation. Such weight-2 patterns exist in a distance-2 code and are the leading source of post-selected logical error.
  • The exact coefficient requires enumerating all pairs of fault locations (choose 2 among 5) and, for each pair, all 15×15 Pauli pairs with their propagation through the intervening Clifford gates, then checking detectability and logical action. This combinatorial enumeration yields a constant c such that: F_logical(p) = 1 − c p^2 + O(p^3)

  • Due to the symmetry and standard construction of this [[4,2,2]] FT state-prep gadget, single-fault contribution vanishes (c_1 = 0), and c is finite and positive.

  • Acceptance probability:

  • The post-selection also rejects many first-order faults, so the acceptance probability P_accept = 1 − a p + O(p^2) with some a > 0. However, the requested quantity is the fidelity of the final logical state conditioned on acceptance, which is governed by the undetected logical-error rate among accepted runs and thus starts at O(p^2).

Let p be the two-qubit depolarizing error probability per CNOT. Then, with post-selection on the ancilla outcome 0 and on all detectable code errors, the logical state fidelity satisfies

\[ F_{\text{logical}}(p) = 1 - c\, p^2 + O(p^3), \]

with no linear term:

\[ \left.\frac{d F_{\text{logical}}}{d p}\right|_{p=0} = 0. \]
  1. Conventions and Units
  • Gates are ideal; noise is only the two-qubit depolarizing channel after each CNOT, with parameter p as defined.
  • Post-selection removes any run with ancilla measurement 1 or with a detectable code syndrome flip.
  1. Final Answer

Final Answer: F_logical(p) = 1 − c p^2 + O(p^3), with no O(p) term. The constant c > 0 depends on the detailed two-fault propagation combinatorics of the given circuit; by construction of this fault-tolerant [[4,2,2]] state-preparation, all single-fault events are detected, so the leading logical infidelity is quadratic in p.
